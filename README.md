# ArcLight Press — Book Projects

This repository contains source files and build scripts for ArcLight Press publications.

## Projects

### POS Systems Guide (`pos-systems-guide/`)
*The Complete POS Systems Guide for Businesses — 2026 Edition*

A comprehensive, 25-chapter reference guide covering POS platforms, payment processing, hardware, security, legal compliance, and implementation strategy. Formatted for Amazon KDP (6" × 9").

**Build:**
```bash
cd pos-systems-guide
pip install weasyprint pymupdf
python render.py
```
